# Contexts Folder

Folder ini digunakan untuk manajemen *state* global aplikasi menggunakan React Context API.

## Isi Folder:
- `AuthContext.jsx`: Mengelola data otentikasi pengguna, status login, dan fungsi terkait login/logout.
- `useAuth.js`: *Custom hook* untuk mempermudah akses ke `AuthContext` di seluruh komponen.
