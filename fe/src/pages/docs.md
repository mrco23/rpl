# Pages Folder

Folder ini berisi komponen-komponen tingkat halaman utama yang dihubungkan langsung ke sistem *routing* (React Router).

## Struktur Folder:
Halaman-halaman biasanya dikelompokkan berdasarkan hak akses atau modul:
- `admin/`: Halaman untuk dashboard admin.
- `verifikator/`: Halaman untuk proses verifikasi dokumen.
- `pendaftar/`: Halaman untuk formulir dan status pendaftaran.
- `public/`: Halaman yang dapat diakses tanpa login (Landing Page, Berita, dll).
- `auth/`: Halaman Login dan Registrasi.
