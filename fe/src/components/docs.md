# Components Folder

Folder ini berisi komponen-komponen UI yang bersifat *reusable* (dapat digunakan kembali) di berbagai halaman aplikasi.

## Prinsip Penggunaan:
- Komponen di sini harus bersifat modular dan sebisa mungkin tidak memiliki ketergantungan langsung pada *state* global yang spesifik, kecuali melalui *props*.
- Digunakan untuk menjaga konsistensi desain di seluruh aplikasi.
- Contoh: Button, Modal, Card, Navbar, Sidebar, dll.
