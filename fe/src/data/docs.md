# Data Folder

Folder ini berisi data statis, konstanta, atau konfigurasi yang digunakan untuk mengisi konten aplikasi tanpa harus mengambil dari API.

## Isi Folder:
- `navigation.js`: Definisi struktur menu navigasi untuk berbagai peran (Admin, Verifikator, Pendaftar).
- `mockData.js`: Data contoh untuk keperluan pengembangan atau *fallback* jika API tidak tersedia.
