# Utils Folder

Folder ini berisi fungsi-fungsi utilitas atau *helper functions* yang sering digunakan di berbagai tempat dalam aplikasi.

## Fungsi Umum:
- Format tanggal dan mata uang.
- Validasi input formulir sederhana.
- Manipulasi *string* atau array.
- Manajemen *localStorage* atau cookie (seperti `token.js`).
